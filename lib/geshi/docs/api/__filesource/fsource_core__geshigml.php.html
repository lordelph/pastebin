14     * GML (Game Maker Language) is a script language that is built-in into Game Maker,
15     * a game creation program, more info about Game Maker can be found at
16     * http://www.gamemaker.nl/
17     * All GML keywords were extracted from the Game Maker HTML Help file using a PHP
18     * script (one section at a time). I love PHP for saving me that bunch of work :P!.
19     * I think all GML functions have been indexed here, but I'm not sure about it, so
20     * please let me know of any issue you may find.
21     *
22     * CHANGES
23     * -------
24     * 2005/11/11
25     *  -  Changed 'CASE_KEYWORDS' fom 'GESHI_CAPS_LOWER' to 'GESHI_CAPS_NO_CHANGE',
26     *     so that MCI_command appears correctly (the only GML function using capitals).
27     *  -  Changed 'CASE_SENSITIVE' options, 'GESHI_COMMENTS' from true to false and all
28     *     of the others from false to true.
29     *  -  Deleted repeated entries.
30     *  -  div and mod are language keywords, moved (from symbols) to the appropiate section (1).
31     *  -  Moved self, other, all, noone and global identifiers to language keywords section 1.
32     *  -  Edited this file lines to a maximum width of 100 characters (as stated in
33     *     the GeSHi docs). Well, not strictly to 100 but around it.
34     *  -  Corrected some minor issues (the vk_f1...vk_f12 keys and similar).
35     *  -  Deleted the KEYWORDS=>5 and KEYWORDS=>6 sections (actually, they were empty).
36     *     I was planning of using those for the GML functions available only in the
37     *     registered version of the program, but not anymore.
38     *
39     * 2005/06/26 (1.0.3)
40     *  -  First Release.
